🚀Changelog

Tracking every iteration of Ling Player

0.5.1.0 2026-05-11 Alpha
✨ New Features
  • Dynamic Gradient Wallpaper for Playback Interface：You can now set a dynamic gradient wallpaper for the playback interface. Find the Wallpaper option under Settings → UI → Playback Interface to configure it
⚡ Improvements
  • Multiple improvements made tothe desktop lyrics feature, enhancing stability and display quality
  • Updated the icons in the sidebar statistics menu
🔧 Bug Fixes
  • Fixed an issue where the popup notification displayed 0 as the song count when adding songs to the playback queue
  • Fixed an issue where the outer visual effect of the album cover was being clipped in the playback interface
0.4.0.0 2026-05-07 Alpha
✨ New Features
  • Folder drag-and-drop support in the tracks page：You can now drag a folder containing tracks directly from File Explorer onto the tracks page to add the folder and automatically start scanning for tracks.
  • Scan progress dialog：Scan progress is now displayed as a standalone dialog window.
🔧 Bug Fixes
  • Fixed an issue where the button spacing in the top-right corner of the playback queue panel was too wide.
  • Fixed an issue where the text and description of the "Image Corner Strength" option did not appear when using English language.
  • Fixed an issue where text in the plugin management page and its settings page did not update after switching languages.
0.3.4.0 2026-05-07 Alpha
🔧 Bug Fixes
  • Fixed an issue where color extraction from grayscale album covers produced incorrect results.
  • Fixed a UI issue where text labels on the vertical axis of the statistics chart could overlap.
  • Fixed an issue where hour tracking in the time period statistics was incorrect.
0.3.2.0 2026-05-05 Alpha
⚡ Improvements
  • Improved the album cover color extraction algorithm, color results are now more accurate.
  • Songs already in the queue can now be set as "Play Next", the queue position will be adjusted automatically.
🔧 Bug Fixes
  • Fixed an issue where krc and qrc format lyrics files could not be loaded or parsed correctly.
0.3.1.0 2026-05-04 Alpha
⚡ Improvements
  • Adjusted the icon size on the left side of the progress bar at the top of the stats page for better visual alignment.
🔧 Bug Fixes
  • Fixed an issue where today's and this week's play time and play counts did not reset correctly when crossing a day or week boundary.
  • Fixed an issue where play time statistics would unexpectedly increase or decrease when manually adjusting the system time.
0.3.0.0 2026-05-03 Alpha
✨ New Features
  • Playback controls can now be always visible. Enable this via the "Settings -> UI" page using the "Play control always visible" toggle.
  • Added image corner strength control. Expand the "Image corner strength" option in "Settings -> UI" to independently adjust the corner radius for song covers, album covers, and artist images.
🔧 Bug Fixes
  • Fixed an issue where Steam Rich Presence would not display according to the format set in the Steam plugin when launching the application.
0.2.1.0 2026-05-02 Alpha
✨ New Features
  • Album list / album details page: Added option to delete album，only available when the album contains no songs.
  • Artist details page Added option to delete artist，only available when the artist has no albums and no songs.
⚡ Improvements
  • Steam plugin is now enabled by default.
  • Tracks shorter than 60 seconds no longer count toward achievements — we want achievements to represent real listening habits.
🔧 Bug Fixes
  • Fixed a crash on startup caused by strict disc/track number non-overlap requirements when multiple identical albums with the exact same artist list existed.(thanks @FunkeymonkeyTTR)
  • Fixed an issue where tracks excluded by the duration filter could not be re-added after adjusting the filter setting and rescanning.(thanks @Nomstuff)
  • Fixed an issue where artist/album artist properties could contain duplicate identical names.
  • Fixed an issue where old cover files were not deleted when updating album or artist artwork.
0.1.1.0 2026/05/01 Alpha
⚡ Optimizations
  • The ""No Lyrics"" text now applies track cover's main color.
🔧 Bug fixes
  • Fixed incorrect play/pause button state in tray menu.
  • Fixed an issue where tray menu could not be closed after opening.
  • Fixed an issue where clicking a song in the list would always play the first song when the action was set to replace the current queue.
0.0.1.0 2026/05/01 Alpha
    First version of Ling Player.
